Este é um repositório para o conjunto de arquivos que representam as
soluções para os respectivos níveis do desafio da Sieve. 
O objetivo é extrair os preços que estão escondidos em cada nível deste
desafio.

Por convenção adotou-se que cada arquivo terá a seguinte nomeação:
levelx.py onde x representa o número do nível em questão.
